# supplier_risk_prediction
# Supplier Risk Prediction Model

## Problem
Procurement teams need to identify high-risk suppliers early to reduce exposure to unreliable partners. This project builds a machine learning model to predict supplier risk based on historical performance and financial data.

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Train/Test Split
5. Model Comparison (Logistic Regression, Random Forest)
6. ROC-AUC Evaluation
7. Business Recommendations

## Deliverables
- Jupyter Notebooks (full lifecycle)
- Clean GitHub repo with modular code
- Visualizations (ROC curve, feature importance)
- Business impact summary

## Business Impact
By flagging high-risk suppliers, procurement can:
- Reduce supply chain disruptions
- Improve vendor selection
- Save costs by avoiding unreliable suppliers
